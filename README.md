# Como-asignar-un-color-de-accent-a-un-progress-Bar-Android

## Se crea el stylo

```
<style name="EcrProgressWhite" parent="Theme.AppCompat.Light">
  <item name="colorAccent">@color/colorWhite</item>
</style>
```

## Se asigna el estilo:

```
android:theme="@style/AppTheme.WhiteAccent"
```
